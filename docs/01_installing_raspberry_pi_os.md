# Installing Raspberry Pi OS Lite (32-bit)

1. Download Raspberry Pi OS Lite (32-bit):  
   https://www.raspberrypi.com/software/operating-systems/

2. Flash the image using Raspberry Pi Imager, balenaEtcher, or dd.

3. Insert the SD card (but do not boot yet).

4. Continue with “Enable SSH and WiFi” in the next document.
